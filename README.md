# 🤖 ResearchBot Pro - AI Business Problem Solver

An intelligent multi-agent system that solves business problems using AI-powered CEO, CFO, and Problem Solver agents.

## ✨ Features

- 🧠 Multi-Agent AI System (CEO, CFO, Problem Solver)
- 📊 Interactive Dashboard
- 📁 File Upload & Analysis
- 🔍 Deep Research Capabilities
- 💼 Business-Focused Insights

## 🛠️ Tech Stack

- Python 3.10+
- Flask / Streamlit
- LangChain
- OpenAI API

## 🚀 Installation

\`\`\`bash
# Clone repo
git clone https://github.com/YOUR_USERNAME/researchbot-pro.git
cd researchbot-pro

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Setup environment variables
cp .env.example .env
# Add your API keys in .env
\`\`\`

## 💻 Usage

\`\`\`bash
python app.py
